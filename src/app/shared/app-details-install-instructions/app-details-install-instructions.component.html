Installation instructions

1. Click to download is available on Arch, Debian Stretch (or later), Endless OS, Fedora 25 (or later), Mageia 6 (or later), and OpenSUSE Tumbleweed

2. If you don't have it already, you'll need to  get Flatpak

3. For Flatpak versions < 0.10.0 it is also recommended to add the flathub repository first. You can  download the repository file and use your distribution's software manager (GNOME Software, KDE Discover, ...) to install it.

4. Click on the INSTALL button

5. Your distibution's software manager will open automatically and will install the app

These command line instructions require Flatpak 0.6.13 or newer. Once installed, applications can be launched from the usual place in your desktop. For more information on how to update and manage your applications, see the command line tutorial.

1. If you don't have it already, you'll need to  get Flatpak

2. Run this on the command line:

 
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install --from {{getDownloadFlatpakRefUrl()}}
